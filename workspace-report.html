  • ✅
    Server Components por defecto
    Solo los componentes que lo necesitan tienen 'use client'. Nav y Footer son Server Components.
  • ✅
    Alias de paths configurado
    El alias ~/ apunta a src/. Evita rutas relativas profundas.
  • ⚠️
    Sin CLAUDE.md en el proyecto
    No existe un archivo CLAUDE.md en la raíz del proyecto. Esto hace que el agente parta sin contexto específico del proyecto en cada sesión y no delega bien las tareas. Recomendado crear uno con descripción del proyecto, convenciones y estado actual.
  • ⚠️
    Sin README.md
    No hay documentación de onboarding. Útil para nuevos colaboradores o para que los agentes entiendan el propósito y estado del proyecto rápidamente.
  • ⚠️
    Rama activa no es main
    Estás en feature/cases-redesign. Si el rediseño está listo, considera mergear a main. Si hay trabajo pendiente, documenta el estado en CLAUDE.md.
  • ℹ️
    Sin loading.tsx ni error.tsx
    Las rutas no tienen loading.tsx con skeletons ni error.tsx como boundary. Para un portfolio es opcional, pero mejora la UX percibida.
  • ℹ️
    Datos estáticos en constants/
    Los case studies están hardcoded en constants/cases.ts. Correcto para un portfolio estático. Si en el futuro se quiere un CMS, habrá que migrar a un repositorio con Supabase o un headless CMS.
Acción recomendada — Crear CLAUDE.md

El problema principal que describes (el agente no responde bien ni delega tareas) se debe casi siempre a la falta de un archivo CLAUDE.md en la raíz del proyecto. Sin él, el agente arranca sin saber qué es el proyecto, qué tech stack usa, cuál es el estado actual, ni cómo dividir las tareas.

El CLAUDE.md debe incluir:

  • Descripción del proyecto: qué es, para quién, propósito.
  • Estado actual: qué está hecho, qué está en curso, qué falta.
  • Tech stack: Next.js 15, React 19, TypeScript, Tailwind.
  • Convenciones: naming, sufijos, dónde va cada cosa.
  • Instrucciones para el agente: cómo dividir tareas, qué agentes usar.
  • Rama activa y próximos pasos: contexto de la rama feature/cases-redesign.

Dime si quieres que lo genere automáticamente basándome en este informe.
